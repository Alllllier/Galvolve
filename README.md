# Galvolve
H5 Galgame 引擎

## 下载 
`git clone git@github.com:Alllllier/Galvolve.git`

## 目标
可以通过工作台快速完成galgame的声音，图片，文字资源的拼装，组成可在Web端运行的Galgame程序。
